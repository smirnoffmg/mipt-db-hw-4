# Домашнее задание 4. Выборка и агрегация данных в MongoDB

Проект для работы с базой данных MongoDB и выполнения запросов на выборку и агрегацию данных.

## Структура проекта

### Основные файлы

- **`hw4.ipynb`** - Jupyter ноутбук с основным кодом
  
### Файлы с данными

Все данные находятся в директории `data/`:

- **`data/theaters.json`** - данные о кинотеатрах
  - Содержит информацию о кинотеатрах: адрес, местоположение и т.д.

- **`data/comments.json`** - данные о комментариях
  - Содержит комментарии пользователей к фильмам: имя пользователя, текст комментария, ID фильма и т.д.

- **`data/movies.json`** - данные о фильмах
  - Содержит информацию о фильмах: название, год выпуска, жанры, режиссеры и т.д.

- **`data/sessions.json`** - данные о сеансах
  - Содержит информацию о сеансах в кинотеатрах

- **`data/users.json`** - данные о пользователях
  - Содержит информацию о пользователях системы

## Использование

1. Запустите MongoDB: `make up`
2. Откройте ноутбук `hw4.ipynb` в Jupyter
3. Выполните ячейки по порядку:
   - Подключение к базе данных MongoDB
   - Выполнение запросов на выборку
   - Построение конвейеров агрегации
   - Визуализация результатов

4. Остановите MongoDB: `make down`

## Примечания

- База данных: `sample_mflix`
- Коллекции: `theaters`, `comments`, `movies`, `sessions`, `users`
- MongoDB запускается в Docker контейнере на порту 27017
- Для работы требуется Python 3.12+ и библиотеки: PyMongo, Pandas, Matplotlib, Seaborn
